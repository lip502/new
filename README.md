# 基于javaweb新闻发布管理系统

#### 介绍
项目技术(必填)
SpringMVC+mybatis+EasyUI+jQuery+JSP
项目描述：

1. 前台新闻阅览

   新闻列表展示、新闻详细、新闻分类、新闻统计、新闻搜索

   发表评论、最新评论文章列表

2. 后台新闻管理

   系统设置：菜单管理、角色管理、修改密码

   用户管理：用户列表

   新闻管理：分类管理、新闻管理、评论管理

   系统日志：日志列表
